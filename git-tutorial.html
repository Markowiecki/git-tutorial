Komendy git

  • git --version -- Wersja Gita
  • git config --global user.name name -- Globalna nazwa
  • git config --global user.email email -- Globalny email
  • git config user.name/email -- Sprawdzenie nazwy/emaila
  • cd .. -- Nawigacja
  • ls -- Lista plików
  • jak dziwny komunikat to SHIFT + : -- później wq + Enter
  • mkdir file.smth -- Utworzenie folderu
  • rmdir file.smth -- Usunięcie folderu
  • touch file.smth -- Utworzenie pliku
  • rm file.smth -- Usunięci pliku
  • Atom file.smth -- Atom otwiera plik
  • git status -- Status Gita
  • git add file.smth -- Dodanie pliku do kolejki
  • git rm --cached file.smth -- Usunięcie pliku z kolejki
  • gti add . -- Dodanie wszystkich plików do kolejki
  • git commit -m "Message" - Dodanie pliku z kolejki do repo z wiadomością
  • git log -- Logi zmian w Gicie
  • git log --oneline -- Log w 1 lini
  • git checkout abc123 -- Sprawdzenie wersji
  • git checkout master -- Powrót do głównego widoku
  • git revert abc123 -- Nowy commit - odwrócenie zmian
  • git reset abc123 -- Reset zmian
  • git reset abc132 --hard -- Twardy reset zmian
  • git branch name -- Tworzy nową gałąź
  • git branch -a -- Na której gałęzi jesteśmy
  • git checkout name -- Przełączenie na inna gałąź
  • git branch -d name -- Usuwanie gałęzi(połączonej)
  • git branch -D name -- Usuwanie gałęzi(nie połączonej)
  • git checkout -b name -- Tworzy nową i przechodzi do niej
  • git merge name -- Dołącza nową gałąź
  • git push www z gita master -- dodanie plikow do GitHub
  • git remote add alias -- Stworzenie aliasa repo
  • git clone wwww -- Tworzenie nowego repo na dysku
  • git remote -v - sprawdzenie gdzie pushować(allias)
